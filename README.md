Tech Stack: HTML5, CSS3, JavaScript, Node.js, MySQL
* Developed a full-stack web platform enabling students of the Technical Education Department to browse, search, and 
apply for jobs and internships through an interactive interface.
* Implemented features for job listing management, student profile creation, and application tracking using Node.js 
backend and MySQL database.
* Designed a responsive and user-friendly front-end with HTML5, CSS3, and JavaScript to ensure accessibility across 
devices for both students and department administrators.
